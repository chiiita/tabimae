# たびまえ｜はじめての海外旅行の準備・現地情報サイト

**公開URL：https://chiiita.github.io/tabimae/**

海外旅行の初心者・不安な人向けに、「渡航前の準備」から「現地での動き」「もしもの対処」まで、
**必要なことを50カ国分まとめた無料の情報サイト**です。広告・課金なし。使いやすさ最優先。

> ターゲット＝はじめて〜数回目の海外旅行で、失敗やトラブルが不安な人。
> コンセプト＝「迷わない・不安をなくす」。

---

## できること（国別・10タブ）
各国ページで次の情報をまとめて確認できます。

1. **準備** — 出発前チェックリスト（チェックは自動保存）
2. **基本情報** — 通貨・チップ・交通・気候・電圧・ネット・水・時差
3. **ビザ・入国** — ビザ要否・事前申請（ESTA等）・パスポート残存・滞在日数
4. **為替** — 円との相場チャート＋現在レート＋かんたん電卓＋早見表（自動取得）
5. **予算の目安** — 食事・水・交通・宿・1日の目安
6. **注意点** — 治安・詐欺・マナー・法律・服装
7. **会話** — 場面別フレーズ（9場面・現地語＋カタカナ・**🔊音声読み上げ**・タップでコピー）
8. **到着後** — 入国〜市内までの「最初の動き」ガイド
9. **緊急時・トラブル対処** — 緊急番号（タップ発信）＋「こんな時どうする」フロー
10. **アプリ** — 配車・地図・翻訳・交通系など

ホームには **国名検索**・**地域別一覧（6地域・50カ国）**・**はじめての海外旅行Q&A／用語集** も。

## 技術
- **ビルド不要のバニラJS**（HTML/CSS/JS）。`index.html` ＋ `data/<国>.js`（国別データ）。
- **データ駆動**：国を増やすときは `data/<id>.js` を足し、`index.html` の `<script>` と `ORDER` に1行追加するだけ。
- **デザイン**：ディープネイビー×ゴールド。アイコンはインラインSVG（Lucide系）。国ヘッダーは国別ランドマークのシルエット（`data/landmarks.js`）。
- **PWA対応**：`manifest.json` ＋ `sw.js` でオフライン閲覧・ホーム画面追加に対応（注意点・フレーズ・緊急番号などはネットなしで閲覧可。為替の最新レートのみ通信）。
- **音声読み上げ**：Web Speech API（端末標準）でフレーズを現地語発音。
- **為替**：Coinbase（即時）→ open.er-api（日次）にフォールバックして自動取得。チャートはTradingView埋め込み。

### ローカルで動かす
```bash
python3 -m http.server 8000   # http://localhost:8000
```
※ 音声・PWA・チェックリスト保存・為替は http(s) 配信時に完全動作します（`file://` 直開きでは一部制限）。

## 注意・免責
- 情報は **2026年6月時点の一般的な目安**です。ビザ・治安・法律・連絡先・為替などは変わります。
- 緊急番号は全50カ国を確認、ビザ・入国は最新情報で再検証していますが、**渡航前には必ず外務省「海外安全ホームページ」や各国の公式情報で最新をご確認ください。**
