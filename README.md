# たびまえ｜旅行前の注意点＋現地フレーズサイト

旅行者向けに「渡航前の注意点（治安・詐欺・マナー・法律）」と「場面別の現地フレーズ」をまとめた、
データ駆動の単一HTMLサイト。**無料サイトで集客 → noteで国別 完全ガイドを販売**するモデル。

## 構成
```
tabimae/
├─ index.html        … アプリ本体（バニラJS・ビルド不要）
└─ data/
   ├─ thailand.js    … タイ
   ├─ korea.js       … 韓国
   ├─ taiwan.js      … 台湾
   └─ usa.js         … アメリカ・ハワイ
```

## コンテンツの柱（各国ページのタブ）
1. 🧭 基本情報（通貨・チップ・交通・気候・電圧・ネット・水・時差）
2. ⚠️ 渡航前の注意点（治安／詐欺・ぼったくり／マナー・文化／法律／服装）
3. 💬 場面別フレーズ（あいさつ／飲食／買い物／移動／緊急）※タップで現地語コピー
4. 🆘 緊急時情報（緊急番号・大使館・パスポート紛失手順）※電話番号はタップ発信
5. 📱 便利アプリ（配車・地図・翻訳・交通IC）

## 国を増やす手順（これだけ）
1. `data/<id>.js` を既存ファイルをコピーして作成（中身を新しい国に書き換え）
2. `index.html` の `<script src="data/...">` に1行追加
3. `index.html` 内 `const ORDER = [...]` に `id` を追加（表示順）

## プレビュー
- `data/*.js` は `<script>` 読み込みなので **ローカルで `index.html` を直接開くだけ** で動く（fetch不使用＝CORS問題なし）。
- ローカルサーバ例: `python3 -m http.server` → http://localhost:8000

## 公開（GitHub Pages）
1. GitHubに新規リポジトリ作成（例: `tabimae`）
2. このフォルダの中身をpush
3. Settings → Pages → Branch: `main` / `/ (root)` で公開
4. 公開URL: `https://<ユーザー名>.github.io/tabimae/`
   - ※ `/Volumes/CIRAGO` はgit working treeが壊れる既知問題あり。pushはユーザー端末側 or 既存の特殊手順で。

## note連携（収益化）
- 各 `data/<id>.js` の `noteUrl: ""` に完全版noteのURLを入れると、国ページ下部の
  「完全ガイド」CTAボタンが有効化される（空のままなら「準備中」表示）。
- 完全版に入れる差分の例: エリア別の詳しい治安、詐欺の具体手口、追加フレーズ、
  モデル準備リスト、おすすめ配車・SIM、両替テクなど。

## 注意
- 治安・法律・連絡先・為替は変動する。**公開前/販売前に最新の公式情報（外務省 海外安全HP等）で要確認。**
- フレーズの発音はカタカナ目安。完全版で音声リンク等を足すと価値が上がる。
