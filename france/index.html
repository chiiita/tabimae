基本情報

通貨・チップ・交通・気候など
通貨
ユーロ（€）
1ユーロ=約170円前後。クレジットカードが広く普及し、少額でもカード可。市場や一部店では現金も用意
チップ
原則不要（任意）
料金にサービス料込み。満足したらカフェで端数、レストランで5〜10%置くと喜ばれる程度
交通
メトロ・TGVが充実
パリは地下鉄・RER・バスが発達。ICカードのNavigoやチケットt+を利用。都市間はTGVが速い
気候
温暖な西岸海洋性
夏は乾燥して過ごしやすく、冬は東京より寒い。年間通じ天候が変わりやすく羽織ものが役立つ
電圧・プラグ
230V / Cタイプ
周波数50Hz。プラグは丸ピン2本のCタイプ（SEも可）。日本の機器は変換プラグと電圧確認が必要
ネット
eSIM / EU圏SIMが便利
Orange・SFRなどのプリペイドやeSIMが入手しやすい。カフェ・ホテルの無料Wi-Fiも一般的
水
水道水は飲用可
硬水なので人により胃に合わないことも。気になればミネラルウォーター（eauを購入）
時差
日本より8時間遅い
夏時間（3月末〜10月末）は7時間遅い。日本が正午のとき冬は午前4時、夏は午前5時

渡航前の注意点

治安・詐欺・マナー・法律
治安・防犯
  • パリの観光地・メトロでは複数人組のスリ集団が活動。混雑時はバッグを体の前で抱える。
  • エッフェル塔・ルーブル・シャンゼリゼ・モンマルトル周辺は特に被害が多い。
  • メトロの乗降時、ドアが閉まる直前にスマホやバッグをひったくる手口に注意。
  • リュックは背負ったままにせず前に抱える。後ろポケットに財布やスマホを入れない。
  • 夜間の北部・郊外の一部地域や人気のない駅は避け、配車アプリを使う。
詐欺・ぼったくり
  • 「署名してください」と寄付の署名を求めながら隙に財布を抜く署名詐欺が有名。
  • ミサンガ（手首に紐を巻く）を勝手に付けて代金を請求する手口。手を出させない。
  • 「地面の指輪を拾った」と渡してきて金を要求する落とし物詐欺に応じない。
  • 正規でない白タクは高額請求のもと。空港から市内はタクシー乗り場かUberを使う。
  • 署名・募金・アンケートを装い近づく集団には足を止めず無視して立ち去る。
マナー・文化
  • 店に入る・出る際はBonjour/Au revoirの一言を。挨拶なしは失礼にあたる。
  • 店員を呼ぶとき大声や指笛はNG。アイコンタクトやS'il vous plaîtで控えめに。
  • レストランは基本的に着席スタイル。相席はせず、急かさずゆっくり食事を楽しむ。
  • 美術館・教会では静かに。教会内では帽子を取り、写真撮影の可否を確認する。
  • 順番待ちの列は守る。割り込みは強く嫌われる。
法律・禁止事項
  • 公共交通の無賃乗車は高額罰金。チケットは下車まで保管し検札に備える。
  • 公共の場での泥酔・路上飲酒は地域により規制。ゴミのポイ捨ても罰金対象。
  • 偽ブランド品の購入・所持は違法で、観光客でも罰金の対象になりうる。
  • ストライキ（grève）でメトロ・鉄道・空港が止まることがある。事前に運行情報を確認。
  • 公共施設や一部の場所での顔を覆う服装の着用が法律で制限されている。
服装
  • パリはシックで落ち着いた装いが好まれる。派手すぎず清潔感のある服装が無難。
  • 高級レストランやオペラ鑑賞はスマートカジュアル以上を意識する。
  • 教会見学では肩や膝を覆う服装を。露出の多い服は避ける。
  • 石畳が多く歩きやすい靴が必須。ヒールやサンダルは歩行に不向き。
  • 天候が変わりやすいので、軽い上着とストールで温度調整できるようにする。

着いたら、まず

入国〜市内までの最初の動き
  1. 入国審査・税関
    パスポート・帰りの航空券・滞在先住所を準備。（ビザ/事前申請：不要（シェンゲン圏／90日以内） / 当面は不要（ETIASは2026年後半に運用開始予定／要確認））
  2. 通信を確保する
    eSIM / EU圏SIMが便利
  3. お金を用意する
    両替は街中が好レートのことが多い。ATMは銀行併設が安心。通貨：ユーロ
  4. 市内へ移動する
    メトロ・TGVが充実　ぼったくり回避はメーターか配車アプリで。
  5. ホテルにチェックイン
    貴重品はセーフティボックスへ。Wi-Fiパスワードを確認。

パスポート紛失時の手順
まず最寄りの警察署で盗難・紛失届（déclaration de vol）を出して証明書を受け取り、その後大使館で渡航書または旅券の発給を申請する。証明写真と身分を示す書類のコピーを用意する
帰国便に間に合わない場合は「帰国のための渡航書」を申請できる
※ 緊急番号・連絡先は変わることがあります。出発前にメモし、現地の最新情報や外務省「海外安全ホームページ」もご確認ください。
こんな時、どうする？
パスポートを失くした
  1. まず警察で紛失・盗難届を出し、証明書をもらう
  2. 日本大使館・総領事館に連絡
  3. 「帰国のための渡航書」か旅券を申請（証明写真・身分証コピーが必要）
盗難・スリにあった
  1. 身の安全を最優先に確保する
  2. 警察で被害届を出す（保険請求に必要）
  3. カード会社へ連絡して利用停止
体調が悪い・ケガをした
  1. 無理せず休む。海外保険のサポートデスクに相談
  2. 薬局（市販薬）か病院へ。重症は救急車（番号は上記）
  3. 受診したら領収書・診断書を保管（保険請求用）
お金が尽きた・カードが使えない
  1. カード会社へ連絡（不正利用や限度額も確認）
  2. 家族から海外送金（Wise等）を受け取る
  3. どうしても困ったら大使館に相談
